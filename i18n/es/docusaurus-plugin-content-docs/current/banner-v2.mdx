# Integração Banners

API para consumo dos banners

## Métodos

### Listar

`GET https://motorv2.infotravel.com.br/integra/api/topVenda`

Método para listar os banners disponíveis. Pare otimizar a pesquisa o método de listar devolve somente os dados básicos do banner. A API vai devolver os resultados paginados, sendo 20 itens por página.

#### Query Parameters

| Parameter      | Type    | Description                                      |
| -------------- | ------- | ------------------------------------------------ |
| `nrPagina`     | integer | Numero da página                                 |
| `tags`         | array   | Uma lista de TAGS                                |
| `idUnidade`    | integer | ID da unidade , usado para filtrar os resultados |
| `idPontoVenda` | Integer | ID do ponto de venda                             |
| `stManual`     | Boolean | Usado para filtros                               |
| `clientSg*`    | String  | Usado para autenticação. (Pedir para infotera)   |
