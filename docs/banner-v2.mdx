# Integração Banners

API para consumo dos banners

## Métodos

### Listar

`GET https://motorv2.infotravel.com.br/integra/api/topVenda`

Método para listar os banners disponíveis. Pare otimizar a pesquisa o método de listar devolve somente os dados básicos do banner. A API vai devolver os resultados paginados, sendo 20 itens por página.

#### Query Parameters

| Parameter      | Type    | Description                                      |
| -------------- | ------- | ------------------------------------------------ |
| `nrPagina`     | integer | Numero da página                                 |
| `tags`         | array   | Uma lista de TAGS                                |
| `idUnidade`    | integer | ID da unidade , usado para filtrar os resultados |
| `idPontoVenda` | Integer | ID do ponto de venda                             |
| `stManual`     | Boolean | Usado para filtros                               |
| `clientSg*`    | String  | Usado para autenticação. (Pedir para infotera)   |

## Montando Links para B2C / B2B

### B2C

Para o B2C existem duas formas de utilizar os dados dos banners para montar pesquisas:

1. **Montando link direto para a tela de pagamento:**

```javascript
let topVenda = rs[0];
// DOMINIO DO B2C é o link em que o B2C roda
// MD5 funçao para criptogragar para MD5
let link = `DOMINIO B2C /checkout?idTopVenda=${topVenda.id}&chave=${md5(topVenda.id).toUpperCase()}`;
```

2. **Montando link de pesquisa para a tela de disponibilidade:**

```javascript
let topVenda = rs[0];
// QUANTIDADE DE PAX.
// por padrão no banner sao sempre 2 pax. Mas é possivel passar outras configurações
let qtPax = 2;
// exemplo de link
// https://linkb2c.com.br/b2c/banner/{idBanner}/{numeroDePax}
let link = `DOMINIO B2C /banner/${topVenda.id}/${qtPax}`;
```

### B2B

Para pesquisar no infotravel fazer um POST/REQUEST para o link do infotravel.

```html
// URL DO INFOTRAVEl - DOMINIO DO INFOTRAVEL ex:
https://demo.infotravel.com.br/infotravel/public/login.xhtml
<form
  action="[URL DO INFOTRAVEL]"
  id="frmTopVenda"
  method="POST"
  name="frmTopVenda"
>
  <input id="topVenda" name="topVendaId" type="hidden" value="[ID DO BANNER]" />
  <button class="btn btn-outline-secondary btn-comprar" type="submit">
    COMPRE AGORA
  </button>
</form>
```

Ou

```javascript
window.open("[URL DO INFOTRAVEL]/public/login.xhtml?topVendaId=54");
```

### PHP

```php
$curl = curl_init();
curl_setopt_array($curl, array(
    CURLOPT_URL => 'http://motorv2.infotravel.com.br/integra/api/topVenda?clientSg=TESTESG',

    CURLOPT_ENCODING => '',
    CURLOPT_FOLLOWLOCATION => true,
    CURLOPT_CUSTOMREQUEST => 'GET',
```
